# User Preferences

Accumulated from feedback over time. Updated after each run where the user responds to a notification.

## Location

- Max 15 minutes walking distance to a grocery store
- Max 15 minutes walking distance to a subway station on the **green line (T13/T14/T17/T18/T19)** or **red line (T13/T14)**
- Listings that don't meet both criteria should be scored down significantly in the area desirability factor

- Areas of extra interest are: Vasastan, Gröndal, Reimersholme, Hornstull, Karlaplan, Gärdet.
